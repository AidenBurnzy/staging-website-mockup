# staging-website-mockup

Clean static website mockup for Perrine Interiors with consistent placeholder copy and placeholder images.

## Run locally

1. Start a local server:

```bash
python3 -m http.server 8000
```

2. Open `http://localhost:8000`.

## Pages

- `index.html` - home
- `staging.html` - staging details
- `showcase.html` - filterable staging showcase
- `consultations.html` - consultation details and inquiry form
- `shop.html` - placeholder furniture cards
- `blog.html` - placeholder article cards
- `team.html` - placeholder team profiles
- `contact.html` - inquiry and home evaluation forms

## Shared assets

- `styles.css` - unified layout, responsive navigation, and component styles
- `script.js` - back-to-top, mobile nav toggle, and showcase filtering

## Notes

- Forms are wired to `mailto:perrinematerials@gmail.com`.
